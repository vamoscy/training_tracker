# 🏁 Half Marathon Training Tracker

A mobile-friendly training tracker for my 9-week half marathon plan leading up to **April 26, 2025**.

**Goal:** Sub 2:00:00 (6.5 mph average pace)

## Features

- ✅ Check off runs as you complete them
- 📍 Today's run auto-highlighted based on current date
- 📊 Overall progress bar across all 9 weeks
- 🔽 Expandable workout details with structure and pacing notes
- 💾 Progress saved locally in your browser (localStorage)
- 📱 Mobile-optimized, works offline once loaded

## Pace Zones

| Zone | Speed |
|------|-------|
| Easy runs | 5.5–5.7 mph |
| Long runs | 5.7–6.0 mph |
| Goal race pace | 6.5–6.6 mph |
| Strides | 7.1 mph |

## Race Day Strategy

- Miles 1–3: 6.3 mph (conservative — let people pass)
- Miles 4–10: 6.5–6.6 mph (work zone)
- Miles 11–13.1: everything left

## Usage

Open `index.html` in any browser. No installation, no dependencies, no internet required after first load.

To update the plan, edit `index.html` and commit — Netlify redeploys automatically.
